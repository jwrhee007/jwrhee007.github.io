Topic 4. Estimating Probabilities from Data

MLE, MAP, Beta 분포, Linear/Logistic Regression, Gradient Descent

Beta 분포 — Prior의 강도와 방향

α, β 값에 따라 분포의 모양이 달라집니다. α=β=1이면 균등, 값이 커지면 중심에 집중됩니다.

α=β=1 (균등) α=β=2 (약한 peak) α=β=10 (강한 peak) α=7, β=5 (비대칭)
MLE vs MAP 수렴 비교 (θ = 0.6 편향 동전)

데이터가 늘어남에 따라 MLE와 다양한 Prior 설정의 MAP 추정값이 어떻게 수렴하는지 보여줍니다.

MLE MAP (α=β=2, 약한) MAP (α=β=101, 강한 잘못된) MAP (α=7,β=5, 정확한)
Linear Regression의 확률적 해석

각 x에서 y는 직선 wᵀx 주변으로 가우시안 분포를 따릅니다. MLE로 w를 학습하면 MSE 최소화와 동일합니다.

y = wᵀx + ε, ε ~ N(0, σ²) MLE → min MSE
Gradient Descent — 학습률의 중요성

등고선 위에서 학습률(α)에 따른 GD 경로를 비교합니다. 너무 크면 발산, 너무 작으면 느린 수렴.

α = 0.01 (너무 작음)
α = 0.1 (적절)
α = 0.5 (너무 큼)
MLE / MAP 와 모델의 연결

핵심 패턴: MAP = MLE + 정규화(Regularization). 가우시안 Prior → L2 정규화.
